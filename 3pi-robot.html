3Pi Line-Following Robot

Role: Embedded Systems Developer

Platform: Pololu 3Pi · AVR Microcontroller · C

📌 Objective

Develop an autonomous robot capable of following a black line on a white track using real-time sensor input, PID control, and pre-programmed race strategies.

🧠 Features Implemented

  • Calibrated 5 infrared reflectance sensors for track detection
  • Implemented line-following using proportional–integral–derivative (PID) control
  • Optimized turning logic and speed adjustments based on track curvature
  • Battery monitoring for runtime decision-making

⚙️ Tools & Technologies

  • C for AVR using Atmel Studio
  • Pololu 3Pi Robot Platform
  • Serial monitor for live debugging

📈 Results

  • Successfully completed test circuits with sharp turns and forks
  • Improved lap time by over 35% through tuning of PID coefficients
  • Demonstrated consistent stability in both high-speed and narrow path sections
